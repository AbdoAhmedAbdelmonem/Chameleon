Singly Linked List

A linked list is a collection of objects called nodes that are stored randomly in memory. Each node consists of two fields:

  • Data: The information stored at that particular address.

  • Pointer: A reference that holds the address of the next node in memory. The last node in the list points to null.

Uses of Linked Lists

Linked lists do not require contiguous memory; nodes can be stored anywhere in memory and linked together, leading to optimized space utilization. The size of the list is limited only by available memory, and there’s no need to declare it in advance. Empty nodes cannot exist in a linked list. Values of primitive types or objects can be stored in a singly linked list.

Advantages of Linked Lists Over Arrays

While arrays are a common data structure for organizing elements in memory, they have several limitations:

  • Fixed Size: The size of an array must be known in advance, which can be restrictive.

  • Inefficient Resizing: Increasing the size of an array is time-consuming and often requires creating a new larger array and copying elements.

  • Contiguous Memory: Arrays require that all elements be stored contiguously in memory, complicating insertion and deletion operations as it requires shifting elements.

In contrast, linked lists offer several benefits:


  • Dynamic Memory Allocation: Linked lists allocate memory dynamically, allowing nodes to be non-contiguously stored in memory and linked via pointers.

  • Flexible Sizing: The size of a linked list is not defined at declaration and can grow as needed, limited only by available memory.

Singly Linked List (One-Way Chain)

A singly linked list is defined as a collection of ordered elements. The number of elements may vary according to the program's needs. Each node in a singly linked list consists of two parts:


  • Data Part: Stores the actual information represented by the node.

  • Link Part: Stores the address of the node’s immediate successor.

The push_front(char v) function inserts a new node at the beginning of a linked list. Here's a summary of its key steps:
  1. Create a New Node: A new node is created with the value v

  2. Link to Current Head: The new node's next pointer is set to the current head of the list, linking it to the existing first node

  3. Update Head: The list's head pointer is updated to point to the new node, making it the new first element in the list.
//Method to insert node at the end of the list

public void push_back(char v) {
    LinkedListNode newNode = new LinkedListNode(v);

    if (head == null) {
        head = newNode;
        return;
    }

    LinkedListNode pointer = head;

    while (pointer.next != null) {
        pointer = pointer.next;
    }

    pointer.next = newNode;
}
The push_back(char v) function adds a new node to the end (or spa "back") of a linked list. Here’s a summary of its key steps:
  1. Create a New Node: A new node is created with the value "v"

  2. Check for an Empty List:

    • If the head is null (meaning the list is empty), the new node is assigned as the head of the list

    • The function then returns, as the insertion is complete

  3. Traverse to the End of the List:

    • If the list is not empty, a pointer (pointer) is initialized to the head of the list.

    • A while loop iterates through the list until it reaches the last node (where pointer.next is null).

  4. Link the New Node: The new node is linked to the last node by setting pointer.next to the new node.

The print() function displays all the values in a linked list. Here’s a summary of its key steps:
  1. Initialize a Pointer: A pointer (pointer) is set to the head of the linked list.

  2. Traverse the List:

    • Inside the loop, the value of the current node (pointer.val) is printed followed by a space.

    • A while loop iterates through the list as long as pointer is not null .

  3. Move to the Next Node: The pointer is then updated to the next node in the list (pointer = pointer.next)
//Method to remove node at a given position

public void remove(int pos) {
    if(!isEmpty()) {
        LinkedListNode pointer = head;
        int counter = 0;
        while (counter < pos - 1) {
            pointer = pointer.next;
            counter++;
        }
        LinkedListNode j = pointer.next;
        pointer.next = j.next;
    }
}
The remove(int pos) function removes a node from a linked list at a specified position. Here’s a summary of its key steps:
  1. Check if the List is Not Empty: The function first checks if the list is not empty using isEmpty(). If the list is empty, it does nothing.

  2. Initialize a Pointer and Counter: A pointer (pointer) is set to the head of the list, and a counter is initialized to 0.

  3. Traverse to the Node Before the Specified Position:

    • A while loop iterates through the list until it reaches the node just before the desired position (pos - 1).

    • The pointer moves to the next node in each iteration, and the counter is incremented.

  4. Remove the Node:

    • After reaching the correct position, the function sets pointer.next to skip the node at the specified position (j), effectively removing it from the list.
