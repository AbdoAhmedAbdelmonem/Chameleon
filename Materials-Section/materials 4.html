    Cloud Computing

    Cloud computing science is the study of designing and managing computing resources over the internet. It explores data storage, information processing, and cloud service delivery

  • DMA

    Data Mining and Analytics

    Data Mining and Analytics is the study of extracting useful information and patterns from large datasets. It explores techniques for data collection, processing, and interpretation

  • DST

    Data Science Tools and Software

    Data Science Tools and Software is the study of technologies and platforms used for data analysis, processing, and visualization. It explores programming languages, machine learning frameworks, and data management systems

  • MLG

    Machine Learning

    Machine Learning (ML) is the study of algorithms and statistical models that enable computers to learn from data and make predictions. It explores supervised and unsupervised learning, neural networks, and pattern recognition

  • RGA

    Regression Analysis

    Regression Analysis is the study of statistical techniques used to examine relationships between variables and make predictions. It explores linear and nonlinear models, correlation, and trend forecasting

  • FAD

    اسعافات اولية

    علم الإسعافات الأولية هو دراسة التقنيات والإجراءات الطبية الأساسية المستخدمة لتقديم الرعاية الطارئة للمصابين قبل وصول المساعدة الطبية المتخصصة. فهو يبحث في طرق التعامل مع الإصابات، والكسور، والنزيف، والحالات الطارئة مثل النوبات القلبية والاختناق.

  • PBD

    تربية حيوانات اليفة

    علم تربية الحيوانات الأليفة هو دراسة رعاية وتغذية الحيوانات المنزلية لضمان صحتها ورفاهيتها. فهو يبحث في سلوك الحيوانات، والاحتياجات الغذائية، والرعاية الطبية، وأساليب التدريب.

     (Elective)
